# Estudo-e-treino-programacao
Repositório criado para que eu possa colocar em práticas meus estudos de programação, conforme eu vou aprendendo nas aulas da faculdade e em outros locais.
Como é um repositório geral, serão apresentados arquivos com linguagens específicas e outro com multiplas linguagens de programação.
O foco é ir trinando sempre para que o uso das linguagens fique cada vez mais fácil com o tempo.
